# Contribuer au projet

## 🚀 Comment contribuer

1. **Fork** le projet
2. **Crée une branche** : `git checkout -b feature/ma-feature`
3. **Commite** : `git commit -m "Ajout de ma feature"`
4. **Push** : `git push -u origin feature/ma-feature`
5. **Ouvre une Pull Request**

## 📊 Standards de code

- Utiliser PEP8 pour Python
- Ajouter des tests pour chaque nouvelle fonctionnalité
- Documenter le code avec des docstrings
